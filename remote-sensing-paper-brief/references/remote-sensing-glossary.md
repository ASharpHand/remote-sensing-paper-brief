# Remote Sensing Glossary

Use this glossary to keep Chinese notes natural and terminologically consistent. Prefer Chinese narrative, keep official names and abbreviations unchanged, and use English in parentheses only when helpful on first mention.

## Remote sensing tasks

| English | Preferred Chinese |
|---|---|
| remote sensing | 遥感 |
| semantic segmentation | 语义分割 |
| object detection | 目标检测 |
| scene classification | 场景分类 |
| change detection | 变化检测 |
| crop mapping | 作物制图 |
| crop yield estimation | 作物估产 |
| land-cover mapping | 土地覆盖制图 |
| land-use mapping | 土地利用制图 |
| paddy rice mapping | 水稻制图 |
| classification | 分类 |
| regression | 回归 |
| time-series classification | 时间序列分类 |
| heterogeneous | 异质的 |
| homogeneous | 同质的 |

## Keep these unchanged when official names

- Dataset names, such as ImageNet, LoveDA, WHU Building Dataset, DOTA, xView.
- Satellite/sensor/product/platform names, such as Landsat, Sentinel-1, Sentinel-2, MODIS, VIIRS, GEDI, Google Earth Engine.
- Model and module names introduced by the paper, such as PSPR, U-Net, DeepLabV3+, Swin Transformer.
- Metric abbreviations, such as OA, PA, UA, F1, IoU, mIoU, mAP, RMSE, MAE, R2.
